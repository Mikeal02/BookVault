# ⚡ Quick Start

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/bookvault.git

cd bookvault
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Application runs at:

```bash
http://localhost:5173
```
